## Test Case Design & Implementation

ในการพัฒนา **Robot Framework Test Automation** 

- ได้มีการใช้ AI ได้แก่ **Claude** เป็นเครื่องมือช่วยในการเขียนโค้ด 
  เช่น:
  - ช่วยสร้างโครงสร้างคำสั่ง Robot Framework  
  - แนะนำ syntax และ keyword ของ SeleniumLibrary  
  - ช่วยปรับปรุงโค้ดให้ถูกต้องและมีประสิทธิภาพมากขึ้น 
  - ช่วยเขียนและปรับปรุงโค้ด
  - ช่วยแก้ไข error และเพิ่มความถูกต้องของ test script
