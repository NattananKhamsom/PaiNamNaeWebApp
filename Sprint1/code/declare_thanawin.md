AI Usage Declaration

การพัฒนา Feature นี้ใช้ AI-assisted development ดังนี้:

-ใช้ ChatGPT สำหรับ:

ช่วยออกแบบแนวคิด Soft Delete

ตรวจสอบ flow ระหว่าง Backend และ Frontend

ช่วยสรุปและจัดทำเอกสาร Changelog

-ใช้ GitHub Copilot สำหรับ:

ช่วยเติมโค้ด Controller / Service

แนะนำ Prisma query

ช่วยเขียนโครงสร้างฟังก์ชัน

ผู้พัฒนาเป็นผู้ออกแบบ logic, ตรวจสอบความถูกต้อง และทดสอบระบบทั้งหมดก่อนใช้งานจริง
